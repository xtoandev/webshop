## Explore Rest APIs

The app defines following CRUD APIs.

### Admin

| Method | Url | Decription | Sample Valid Request Body | 
| ------ | --- | ---------- | --------------------------- |
| GET   | /api-admin | Get all admin |  |
| GET  | /api-admin/{adminID} | Get admin by ID |  |
